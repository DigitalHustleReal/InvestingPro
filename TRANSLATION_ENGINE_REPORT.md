- **Capabilities**: Translates English HTML -> Hindi/Tamil/Marathi HTML.

## 🖱️ 2. THE "TOGGLE" WORKFLOW
Instead of a "client-side widget" (which hurts SEO), I built a **Generator Workflow**:
1.  Open any English Article.
2.  Select **"Hindi"** (or Tamil, Marathi, etc.) in the Inspector.
3.  Click **"Go"**.
4.  **Result**: The system generates a **Brand New Article** (Draft) in that language.
5.  **Benefit**: You now have a unique URL (e.g., `best-sip-plans-hi`) that **ranks on Google India** for Hindi queries.
