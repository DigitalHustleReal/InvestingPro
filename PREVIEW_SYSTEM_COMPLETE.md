## 🚀 **HOW TO USE:**

### **From Article Editor:**
1. Go to: `/admin/articles/new`
2. Type a title: "Best Credit Cards 2026"
3. Write some content in markdown
4. Click **"👁️ Preview"** button (in inspector)
5. **Preview modal opens!**

### **What You'll See:**
- **Article tab:** How it looks on the website
- **SEO tab:** How it appears in Google/social
- **Device switcher:** Test mobile/tablet/desktop
- **Live updates:** Changes reflect in real-time!
