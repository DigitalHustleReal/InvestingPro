# Populate Products (Safe Mode)

This workflow enables you to automatically fill the database with real-world financial products (Credit Cards, Loans, Insurance, Mutual Funds) while strictly avoiding copyright issues by using pre-generated, safe 3D assets for images.

## Prerequisites
- Supabase Database must be reachable.
- `GOOGLE_GEMINI_API_KEY` must be set in `.env.local` for the data scraping (facts are safe).
- Safe placeholder images must exist in `public/assets/products/` (Generated by Agent).

## Steps

1.  **Run the Safe Population Script**
    This script fetches the latest 2026 specs for defined products using Gemini but applies your custom, safe 3D assets (`generic_credit_card.png`, etc.) instead of downloading logos.

    ```bash
    // turbo
    npx tsx scripts/populate-safe-products_v2.ts
    ```

2.  **Verify Data**
    Check the Admin Panel or the Database to see the new products.
    - Go to `/admin/products`
    - Verify they have the new "Safe" placeholders.

## Customization
To add more products, edit `scripts/populate-safe-products_v2.ts` and add them to the `SAFE_PRODUCTS_TO_ADD` array.
