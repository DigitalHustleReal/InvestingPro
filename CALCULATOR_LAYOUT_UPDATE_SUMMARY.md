# Calculator Layout Update Summary

## Status: In Progress

All calculators are being updated to use the consistent side-by-side layout pattern, removing percentile charts and replacing with year-by-year breakdowns.

## Pattern Applied

### Layout Structure:
1. **Top Row (2 columns):**
   - Left: Compact input card with stacked sliders
   - Right: Results card with stats and summary

2. **Bottom Row (2 columns):**
   - Left: Growth/Projection chart (larger area chart)
   - Right: Year-by-year breakdown table with insights
