## Quick Reference: Template Checklist

Before publishing, ensure:

- [ ] Title is decision-focused (not educational)
- [ ] Introduction identifies problem/need
- [ ] Quick answer section (top 3 at a glance)
- [ ] Detailed recommendations (3-5 options)
- [ ] Comparison table included
- [ ] "How to Choose" section
- [ ] Clear CTAs with affiliate links
- [ ] FAQs (5-7 questions)
- [ ] SEBI/RBI disclaimers where needed
- [ ] Indian context (currency, examples)
- [ ] Honest about fees/limitations
- [ ] Estimated value/returns shown
- [ ] Eligibility clearly stated

---

## Template Customization Guidelines

1. **Always Start with User Problem**
   - Don't start with "What is X?"
   - Start with "If you need X, here are the best options"

2. **Use Real Numbers**
   - ₹20,000/month spending
   - ₹5,000/month SIP
   - ₹10 lakh loan

3. **Show Value, Not Just Features**
   - "Earn ₹6,000/year" not just "5% cashback"
   - "Grow to ₹10 lakh in 10 years" not just "12% returns"

4. **Be Honest About Trade-offs**
   - "Higher rewards but ₹5,000 annual fee"
   - "Better returns but higher risk"

5. **Include Clear Next Steps**
   - "Apply Now" with affiliate link
   - "Invest Now" with affiliate link
   - "Get Quote" with affiliate link
