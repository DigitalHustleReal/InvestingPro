## 🚀 How to Use

### Step 1: Access the AI Writer
1. Go to **Admin Dashboard** (`/admin`)
2. Click the **"AI Generator"** tab
3. You'll see the new Writesonic-style interface

### Step 2: Choose Content Type
Click on any content type icon:
- **Blog Post** - For full articles
- **Headlines** - For title variations
- **FAQ** - For question sections
- **Expand** - To enhance existing content
- **Rewrite** - To improve content
- And more...

### Step 3: Enter Your Input
- **For new content**: Enter your topic (e.g., "Best SIP Plans for Retirement")
- **For expand/rewrite**: Paste your existing content

### Step 4: Configure Options

**Basic Tab:**
- **Keywords**: Comma-separated SEO keywords
- **Tone**: Choose writing style
- **Category**: Select content category
- **Word Count**: Target length (for blog posts)

**Advanced Tab:**
- **Framework**: Choose content structure (AIDA, PAS, etc.)
- **Language**: Select language

### Step 5: Generate
Click **"Generate Content"** and wait for AI magic! ✨

### Step 6: Use Generated Content
- **Copy** - Copy to clipboard
- **Save as Draft** - Save directly to CMS
- **Review SEO Score** - Check optimization level

## 📝 Content Type Examples

### Blog Post
```
Topic: "Best Credit Cards for Travel in India"
Keywords: credit cards, travel rewards, miles
Framework: Comparison
Tone: Professional
Word Count: 2000

→ Generates: Full article with intro, comparison table, pros/cons, conclusion
```

### Headlines
```
Topic: "SIP Investment Guide"
Keywords: SIP, mutual funds, systematic investment

→ Generates: 10 headline variations like:
- "10 SIP Strategies That Could Transform Your Retirement"
- "The Complete SIP Investment Guide for Beginners in 2024"
- "How to Start SIP: A Step-by-Step Guide to Systematic Investing"
```

### FAQ
```
Topic: "Credit Card Benefits"
Category: credit-cards

→ Generates: 8-10 FAQs with detailed answers
```

### Expand Content
```
Existing: "SIP is good for long-term investing."
→ Expands to: Comprehensive explanation with examples, statistics, benefits
```

### Rewrite Content
```
Original: "This product is good."
Tone: Professional
→ Rewrites to: "This product offers comprehensive features that align with modern investment strategies..."
```

## 🎯 Best Practices

### 1. **For Blog Posts**
- Use specific, detailed topics
- Include 3-5 relevant keywords
- Choose appropriate framework (AIDA for persuasive, How-To for tutorials)
- Target 1500-2000 words for SEO

### 2. **For Headlines**
- Keep topic focused
- Include primary keyword
- Generate 10 variations, test different angles

### 3. **For SEO Optimization**
- Paste existing content
- Add target keywords
- Review SEO score (aim for 80+)
- Implement recommendations

### 4. **For Content Expansion**
- Paste short content
- Set higher word count target
- AI will add examples, statistics, details

## 📊 SEO Scoring

The AI writer automatically calculates SEO scores:
- **80-100**: Excellent ✅
- **60-79**: Good ⚠️
- **Below 60**: Needs improvement ❌

**Factors considered:**
- Keyword presence
- Content length
- Heading structure
- Readability

## 🆘 Troubleshooting

### "Error generating content"
- Check your OpenAI API key in `.env.local`
- Verify internet connection
- Try with simpler topic

### "Low SEO Score"
- Add more keywords
- Increase content length
- Add more headings
- Improve readability

### "Content not saving"
- Check Supabase connection
- Verify article permissions
- Review browser console for errors
