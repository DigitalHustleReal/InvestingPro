## 🚀 HOW TO USE EVERYTHING

### **1. Design System**
```tsx
// Colors
className="bg-primary-600 text-white"           // Teal brand
className="bg-stone-50 text-stone-900"          // Neutral
className="bg-success-700 text-white"           // Green (gains)

// Typography
className="font-sans"                            // Inter (UI)
className="font-serif"                           // Source Serif 4 (editorial)
className="font-mono"                            // JetBrains Mono (numbers)

// Spacing
className=p-6 md:p-8"                           // Card padding
className="py-16 md:py-24"                       // Section spacing
```

### **2. Currency Formatting**
```tsx
import { formatINR, formatGainLoss } from '@/lib/utils/currency';

{formatINR(500)}                                 // ₹500
{formatINR(100000)}                              // ₹1,00,000
{formatINR(5000000, { compact: true })}          // ₹50L

const { formatted, color, icon } = formatGainLoss(5000);
<span className={color}>{icon} {formatted}</span> // ▲ +₹5,000 (green)
```

### **3. Compliance**
