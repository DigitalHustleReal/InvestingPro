File: lib/research/keyword-researcher.ts (550 lines)

DATA SOURCES (5):
1. Google Suggest (autocomplete)
2. SerpApi Related Searches
3. People Also Ask questions
4. Competitor analysis (via SERP)
5. LSI keyword database

ANALYSIS ENGINE:

Difficulty Estimation (0-100):
• Analyzes competitor authority
• Checks high-auth domains in top 10
• Long-tail adjustment (-10 to -15 points)
• Algorithm: base 50 + authority×5 - tail_adjustment

Intent Classification (4 types):
• Informational (how, what, why, guide)
• Commercial (best, top, review, vs)
• Transactional (buy, price, cheap, deal)
• Navigational (login, website, official)

Opportunity Scoring (0-100):
Formula: 100 - (difficulty × 0.4) + intent_bonus + tail_bonus
• Low difficulty = high opportunity
• Intent bonus: transactional(20), commercial(15), info(10)
• Long-tail bonus: 4+ words(+15), 3 words(+10)

Keyword Clustering:
• Groups by semantic similarity
• Recommends content types per cluster
• Ranks by total opportunity score

OUTPUTS:
✓ Primary keyword analysis
✓ Related keyword clusters
✓ Long-tail opportunities (difficulty < 40)
✓ Content gaps from SERP
✓ Recommended topics
✓ Search intent classification
✓ Competition level

USE CASES:
→ Content strategy planning
→ Low-competition keyword discovery
→ Topic prioritization
→ SEO gap analysis
